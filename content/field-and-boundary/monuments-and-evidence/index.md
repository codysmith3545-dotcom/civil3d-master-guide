> **TL;DR**
> 1. **Found monuments control over measurements** when they're consistent with the original survey. The order of importance: original/undisturbed monument → relocated/disturbed monument → original lines marked → adjoiner calls → measured distances → directions → quantities.
> 2. Document every monument: type, condition, photo, position, agreement with deed/plat. The narrative explaining the choices is the survey.
> 3. Indiana law requires registered surveyors to file a **Surveyor's Report** (IC 36-2-12-14) for original surveys; many practitioners file for retracement also.
