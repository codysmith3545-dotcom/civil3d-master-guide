> **TL;DR**
> 1. Run the same five-stage checklist on every ALTA: research, monumentation, fieldwork, drawing, certification + electronic deliverables.
> 2. Most failed reviews come from one of three things: missing Schedule B-2 plotting, an edited certification, or a stale title commitment.
> 3. Treat Table A items as line items: each one needs corresponding fieldwork, drawing content, and certification text.

This is a practical pre-flight list. It is organized by phase, not by section number; cross-reference to the 2021 standard before signing.

## 1. Research

- [ ] Title commitment in hand, dated within the standard's currency window. Verify the effective date.
- [ ] All Schedule B-2 documents legible. Re-request unreadable copies before fieldwork starts.
- [ ] Record deeds for the subject and all adjoiners pulled.
- [ ] Recorded plats and any prior surveys of the parcel located.
- [ ] Section corner records (county surveyor, GLO, BLM) pulled where the parcel ties to PLSS.
- [ ] FEMA FIRM panel identified, panel number and effective date recorded.
- [ ] Zoning report received (if Table A item 6 is in scope).
- [ ] Utility plans received from the client (if item 11 is in scope).
- [ ] Right-of-way plans received from the controlling agency (if item 17 is in scope).

## 2. Monumentation and corner analysis

- [ ] Existing monuments recovered, photographed, and described (size, material, cap stamping, condition, position relative to controlling lines).
- [ ] Section corners and PLSS controlling corners tied where applicable.
- [ ] Monuments to be set are identified before mobilization (legible cap, surveyor identification, durable material as required by Section 3.B of the standard).
- [ ] Adjoiner monuments recovered to test for harmony.
- [ ] Pincushion situations documented and resolved with a written rationale; do not silently choose one of multiple monuments.

## 3. Fieldwork

- [ ] Control network established with redundant ties; horizontal and vertical control documented.
- [ ] Boundary corners observed redundantly.
- [ ] Improvements relevant to the boundary located (fences, walls, drives, signs, hedge lines, paved edges encroaching, etc.).
- [ ] Schedule B-2 exception items located on the ground or noted as not locatable with a reason.
- [ ] Utility evidence located per item 11 scope.
- [ ] Field notes and photographs captured for any unusual evidence.
- [ ] Datum and basis-of-bearings recorded.

## 4. Drawing (plat or map)

- [ ] Title block with project name, address, owner, surveyor, date, scale, sheet number.
- [ ] North arrow with the basis-of-bearings statement.
- [ ] Vicinity map.
- [ ] Legal description (matching the title commitment) shown verbatim and the surveyor's measured description if different.
- [ ] All monuments shown with found/set tags and descriptions.
- [ ] All boundary courses with bearing and distance, and any record-vs-measured differences called out.
- [ ] All Schedule B-2 items keyed by exception number, plotted or noted "not plottable" with a reason.
- [ ] Easements (record and apparent) shown with width, source document, and book/page or instrument number.
- [ ] Improvements shown.
- [ ] Encroachments and gaps/overlaps with the adjoiners highlighted.
- [ ] Setbacks per the zoning report (if item 6).
- [ ] Flood zone information per item 3.
- [ ] Gross area per item 4 (and net if requested).
- [ ] Standard ALTA/NSPS notes block.
- [ ] Legend, scale bar, sheet index if multi-sheet.

## 5. Certification and notes

- [ ] The certification block uses the **exact 2021 NSPS language**. Do not paraphrase.
- [ ] Recipients listed in the certification match the agreed-upon list.
- [ ] Surveyor's seal and signature applied to the signed deliverable in the manner required by state law.
- [ ] Date of last fieldwork and date of plat both shown.
- [ ] Table A items selected are listed and reflected in plat content.
- [ ] State minimum technical standards are also satisfied (Indiana surveyors must also comply with 865 IAC 1-12).

## 6. Electronic deliverables

- [ ] PDF/A signed copy delivered.
- [ ] CAD file delivered if scope requires it (DWG version negotiated with the client).
- [ ] Coordinate file (PNEZD or similar) for monuments set and found, if requested.
- [ ] Photographs of monuments archived per office retention policy.

## QA before signing

- Two independent reviewers walk the plat against the title commitment Schedule B-2 line by line.
- A second reviewer verifies the certification is the unedited 2021 language.
- Cross-check the legal description on the plat against the commitment word for word; flag any deviations to the title officer in writing.
