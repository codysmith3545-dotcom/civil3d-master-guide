> **TL;DR**
> 1. Civil/survey work in Indiana sits in three layers of authority: **state** (INDOT, IDEM, PLA), **county** (county engineer, county surveyor, drainage board), **municipality** (city/town engineer, plan commission).
> 2. Many counties are MS4 communities (IDEM Rule 13). Most cities have adopted **design standards manuals** that exceed state minimums.
> 3. This guide focuses on **Marion County** (Indianapolis area) and the **7 surrounding "donut" counties**: Hamilton, Hancock, Shelby, Johnson, Morgan, Hendricks, Boone.

## Working with this section

Each county page lists:
- The county engineer / county surveyor / drainage board / plan commission
- Code of ordinances link (subdivision control, drainage, zoning)
- GIS portal
- Submittal/permit landing page
- Adopted design standards manual (if separate from ordinances)

Each municipality page lists:
- Engineering / public-works contact
- Code of ordinances link
- Adopted design standards manual link
- Plan-review checklist link
- GIS portal

> **Verify before relying.** Jurisdiction websites and standards change. Each page records the date last verified. The companion script `scripts/verify-jurisdiction-links.mjs` runs weekly; flagged links should be the first thing fixed.
