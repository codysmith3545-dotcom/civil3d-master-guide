> **TL;DR**
> 1. **Dynamo for Civil 3D** is a node-based visual programming tool for repeatable workflows — assign elevations, batch-create objects, manipulate corridors.
> 2. Strong for one-off scripts and parameterized workflows that don't justify a .NET plugin.
> 3. Civil 3D ships with a **Dynamo Player** for non-author users to run pre-built `.dyn` graphs from a panel.
