> **TL;DR**
> 1. **View frame groups** lay rectangular frames along an alignment to define each plan/profile sheet's extents.
> 2. **Match lines** are auto-placed between adjacent view frames; their labels reference adjoining sheets.
> 3. The **Create Sheets** wizard turns the view frames into a Sheet Set with viewports already aimed at each frame.
