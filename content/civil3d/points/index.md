> **TL;DR**
> 1. **Description keys** match point raw descriptions (e.g., `EP*`) to a point style, label style, layer, and full description. Set them up once per company.
> 2. **Point groups** filter points by criteria (number range, raw description, elevation, etc.) and drive what you see, plot, and export.
> 3. The default point file format is **PNEZD (comma-delimited)** — Point, Northing, Easting, Z, Description. Know how to import/export it from memory.
