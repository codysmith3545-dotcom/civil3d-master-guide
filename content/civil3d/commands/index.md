> **TL;DR**
> 1. Civil 3D commands break down by object type. Most start with a verb prefix: `Create…`, `Edit…`, `Add…`, `Import…`, `Export…`.
> 2. Use the [shortcuts page](shortcuts.md) for keyboard aliases and the [command-line cheatsheet](command-line-cheatsheet.md) for what to type at the prompt.
> 3. Symptoms-first lookup is in the [chat](/chat) and on each command page's `symptoms:` frontmatter.
